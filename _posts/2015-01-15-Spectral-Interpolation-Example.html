One such application is in the discrete representation of sophisticated functions in multiple dimensions. Instead of interpolating between sampled values of the function in real space, this approximation can be done in DFT space using a technique called **spectral interpolation**. This post describes the technique and gives examples of its use in 1D and 2D. ## 1D Interpolation First we write the original function: $$ \begin{equation} f(x) = -0.75sin(x) + 0.25cos(3x) \end{equation} $$ The $$ f(x) $$ is periodic over the domain $$ 0 \leq x \leq 2\pi $$ for convenience. If this function were non-periodic the DFT method would be less effective, as the DFT assumes periodicity. In this case the function should be mirrored over the y-axis so that it would become periodic over the domain $$ -2\pi \leq x \leq 2\pi $$.
