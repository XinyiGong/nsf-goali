These orientations are shown in the following image: {% include ContentManager.html content=page.cal %} Orientations for the validation microstructures were randomly assigned from a preselected set of 4800 orientations in the hexagonal triclinic fundamental zone. In **Set 1** these orientations spanned the entire face except that only one orientation was selected from the $ \Phi = 0 $ edge.
